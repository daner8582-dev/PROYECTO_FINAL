## SIMULADOR DE CRISIS ECONOMICA FAMILIAR
## DESCRIPCION:
Este proyecto fue desarrollado para la materia de Programación Web I.
La página web permite simular diferentes problemas relacionados
con el contexto actual, como:

- abastecimiento de carburantes
- aumento de precios de alimentos
- incremento del costo de transporte

El objetivo es mostrar resultados dinámicos mediante HTML,
CSS y JavaScript.
---
## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- DOM
- Git y GitHub
---
## Funcionalidades

### Simulador de carburantes
Permite calcular cuántos días durará una reserva de combustible.

### Simulador de alimentos
Permite calcular el aumento del gasto familiar debido al incremento de precios.

### Simulador de transporte
Permite calcular el costo adicional generado por desvíos y rutas más largas.
